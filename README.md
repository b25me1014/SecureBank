# 🔐 SecureBank System (C Project)

A modular console-based banking system developed in C, designed with scalability, validation, and future multi-user support.

---

## 🚀 Features

- Deposit & Withdraw System
- Input Validation (robust)
- Clean Menu-driven UI
- Modular Architecture
- Integration-ready system design

---

## 🧠 System Architecture

The project follows a **modular design**, where each functionality is separated into independent modules:

- `main.c` → Core controller (integration & flow)
- `validation.c` → Input validation system
- `account.c` → Account management (create/search/update)
- `transaction.c` → Deposit & Withdraw logic
- `history.c` → Transaction logging system
- `ui.c` → User interface & navigation
